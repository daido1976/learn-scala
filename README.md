## Setup

```sh
# Require Setup IDE after git clone.
$ scala-cli setup-ide .
# Reload the window of IDE may be necessary.
```

## Run

```sh
# e.g.
$ scala-cli hello
# With Command line arguments
$ scala-cli hello -- taro shota
```
